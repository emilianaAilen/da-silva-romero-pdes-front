# Asesor Personal de Compras ( integracion con API de MELI)

## Requisitos Previos
- Node version >= 16

## Para correr la App
- `npm install`
- `npm start`
Podes usar yarn si preferis

### Docker
`docker-compose up`

Abrir [http://localhost:3000](http://localhost:3000) para ver en el navegador.

## Para correr tests
- `npm test`
